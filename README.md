### 🦀 Why Rust?

> *"C-level performance. Zero garbage collector. Memory safety enforced at compile time."*

| Principle | In Practice |
|-----------|-------------|
| 🔒 **Ownership** | One owner per value — freed deterministically when it leaves scope, no GC needed |
| 📦 **Borrowing** | One mutable *or* many immutable refs — race conditions impossible by design |
| ⏳ **Lifetimes** | Compiler ensures refs never outlive their data — no dangling pointers, ever |
| ⚡ **Zero-cost** | All safety checks at compile time — no runtime overhead, no tradeoffs |
